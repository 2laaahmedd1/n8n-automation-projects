# University Scheduling Automation (n8n) 🚀
This is my first automation project! I built this workflow using **n8n** to streamline university scheduling and data management for credit hours. It’s a work in progress, and I’m continuously improving it.

 * **Data synchronization** between sources.
 * Built with **low-code logic** for maximum efficiency.

 * **JSON** (Data Structure)
